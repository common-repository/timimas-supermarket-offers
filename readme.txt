== Description ==

Use this widget to get all current offers from greek supermarket. The widget connects to timimas.gr to get all current offers and best prices.

* Get's the latest offers from timimas.gr

Here's a link to our website [timimas.gr](http://timimas.gr/ "Σύγκριση τιμών Supermarket").

== Installation ==

1. Upload the files to the /wp-content/plugins/timimas-supermarket-offers/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
